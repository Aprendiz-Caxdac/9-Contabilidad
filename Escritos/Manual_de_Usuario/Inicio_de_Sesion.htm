Inicio de Sesión

Para iniciar la sesión debe contar con usuario y contraseña asignado por el área de sistemas. El password para esta aplicación es el mismo utilizado para el acceso a la estación de trabajo, el usuario ingresa por internet al portal Citrix:

Mediante red interna: Citrix Workspace

Mediante Wifi: NetScaler AAA

Una vez llenado este requisito, le despliega las aplicaciones asignadas de acuerdo a su perfil de usuario. Debe dar doble clic sobre la aplicación que desea abrir.

Esto abrirá la aplicación escogida, en esta debe escribir el usuario y contraseña.

Si desea trabajar en una base de datos diferente de BD Producción, activar el checkbox de "Cambiar Base de Datos", luego escoger de la lista desplegable la Base de Datos en la que desea trabajar.

Por último, clic en "Aceptar", si todo está correcto podrá ingresar a la aplicación.
