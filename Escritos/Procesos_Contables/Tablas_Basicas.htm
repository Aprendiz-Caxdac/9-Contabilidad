Cuentas de Transmisión Superfinanciera

Esta tabla contiene las cuentas a nivel de subcuenta (Catalogo con fines de supervisión Superintendencia Financiera), permite construir adecuadamente el archivo plano para transmitir a la Supe financiera, el usuario debe cargar en esta tabla las cuentas para la elaboración del archivo plano, cada vez que se cree una cuenta en la tabla de catálogo de cuentas a nivel cuenta y subsecuente se debe actualizar en esta tabla.
